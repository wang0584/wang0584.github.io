Notes About HTML Tags

5 HTML Tags Name and Description

Tag Name "html"
is the next and most important tag on our page, anything we add to the page must be inside opening "html"and closing "/html"
Tag Name "title"
is the title of the webpage which will be displayed on top of the tab in a browser. Not on the page.
tag Name "body"
is the main body of the document, this is what we see in our browsers. All the content of the web page must be added inside the opening "body" and closing "/body" tags.
Tag Name "nav"
the navigation of the page, this will contain the links to other pages of the website
Tag Name "main"
is the primary content of the page
